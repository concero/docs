# Routing
:::info
This functionality is still under development but soon	will be available for public use.
:::

Routing is an off-chain service that computes the most optimal transaction routes in terms of speed and cost. At the moment, the service works with Uniswap v2 and v3 on Ethereum, Polygon, Arbitrum, and Avalanche.

## Supported DEXes
- Uniswap v2
- Uniswap v3

:::note
We are soon bringing support for more DEXes in the nearest future.
If you'd like to see a specific DEX supported, please [let us know](mailto:support@concero.io).
:::
